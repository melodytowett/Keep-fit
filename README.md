# Flask Blog

A blog that allows writers to create and share articles. Users can also comment on articles.

## Technologies Used

The following languages have been used on this project:

- HTML & CSS
- Python
- Flask
- PostgreSQL
- Heroku

## Live Demo

[Live demo](https://flask-blog-xxiv.herokuapp.com/)

## Project setup and instructions

Download or clone the source code from [GitHub](https://github.com/davidkiama/Flask-blog.git)

Download and install any stable version of Python.[Python 3.7.3](https://www.python.org/downloads/)

Donwload pip and install it. [pip](https://pip.pypa.io/en/stable/)

Download and install PostgreSQL. [PostgreSQL 9.6.1](https://www.postgresql.org/download/)

Create a database and set the DATABASE_URI environment variable to point to it.

Create a virtual environment. [virtualenv](https://virtualenv.pypa.io/en/stable/)
`virtualenv venv`

Install the required packages using pip.
`pip install -r requirements.txt`

Run the application.
`python manage.py`
